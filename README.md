goLean 

-- Professional MERN Fitness App --
Empowering the Philippines Fitness Journey through Clean Engineering.

goLean is more than just a workout tracker—it’s a high-performance, production-ready fitness ecosystem built for scalability. This project demonstrates advanced Full-Stack JavaScript development, focusing on modular architecture, security, and world-class developer experience.

-- The Tech Stack --
Mobile Engine: React Native (Expo) - Modular UI & Native Performance.
Backend Core: Node.js & Express - Robust RESTful API with strict MVC separation.
Database: MongoDB & Mongoose - Efficient, scalable data mapping.
Security: State-of-the-art JWT (JSON Web Token) authentication with Bcrypt hashing.
Architecture: Clean separation of concerns (Services, Controllers, Screens, Context).
